## Temporary Placeholders

Until you add actual photos, the images will use:
- Gradient backgrounds with destination names
- Or stock photos from unsplash.com

## Recommended Image Specs

- **Format**: JPG or WebP
- **Dimensions**: 800x600px minimum (1200x900px ideal)
- **File size**: Under 200KB each (optimized)
- **Aspect ratio**: 4:3 or 16:9
- **Quality**: High resolution, well-lit, professional

## How to Add Images

1. Save your images to `/public/hero/` folder
2. Name them exactly as listed above
3. Images will automatically appear on the website
4. No code changes needed!

## Using Your Own Photos

If you have photos from client trips or destinations:
1. Get permission if photos include people
2. Ensure photos are high quality
3. Optimize before uploading (compress)
4. Use descriptive names
