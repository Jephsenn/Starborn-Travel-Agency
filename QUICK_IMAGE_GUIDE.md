## Easy 3-Step Process

### Step 1: Get Sara's Photo (5 minutes)
1. Find a good headshot photo
2. Crop to square (use Canva.com if needed)
3. Save as `sara-headshot.jpg`
4. Drop in `/public` folder

### Step 2: Collect Destination Photos (30 minutes - 1 hour)
1. Go to Unsplash.com
2. Search for "cruise ship", "disney castle", "caribbean beach", etc.
3. Download high-quality versions
4. Rename according to IMAGE_GUIDE.md

### Step 3: Upload to Website (5 minutes)
1. Create `/public/hero/` folder
2. Copy all images there
3. Refresh website - done!

## Pro Tips

### Image Optimization
- Download largest size from Unsplash
- Use TinyPNG.com to compress before uploading
- Aim for 150-250KB per image

### Quick Edits
- Canva.com - Free, easy cropping/resizing
- Photopea.com - Free Photoshop alternative
- Remove.bg - Remove backgrounds from headshots

### Testing
1. Add images one at a time
2. Refresh browser after each upload
3. Check on mobile devices
4. Verify fast loading speeds
